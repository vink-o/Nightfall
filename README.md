# Nightfall

A modular C++ application featuring a secure Authentication System and a custom File-based Database. This project focuses on software architecture, header/source separation, and data persistence.

### Key Features
* **Modular Architecture:** Full separation of concerns using `.h` and `.cpp` files.
* **Authentication System:** Secure user login and registration logic.
* **Custom Database Engine:** A file-system-based approach to storing user profiles and game data.
* **Scalable Design:** Built to be the foundation for a larger RPG ecosystem.

### Technical Details
* **Language:** C++17
* **Concepts:** Header Guards, Modular Programming, File I/O, Encapsulation.
